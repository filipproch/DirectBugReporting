An Android library to help you get error reports from your users

Support for:
  • Github issues
  • Redmine

Usage

Report dialog

To show report dialog, in which user can fill issue details use this code:

for Github


GithubService service = new GithubService("RepositoryOwner","RepositoryName");
service.create(null);//always null, url is defined internally
IssueDialog dialog = new IssueDialog.Builder(context)
.setService(service)
.show();

for Redmine

int projectId = 1; //your redmine project ID
RedmineService service = new RedmineService(projectId);
service.create("http://yourredmineurl/");
IssueDialog dialog = new IssueDialog.Builder(context)
.setService(service)
.show();

Manual report

Both methods service.auth(...) and service.report(...) return Bundle object. It contains information about the request like if it was successful. These options are :

  • ReportingService.ACTION_SUCCESS - boolean, always in Bundle
  • ReportingService.RESPONSE_MESSAGE - String, included if there is message in response from server
  • ReportingService.RESPONSE_ISSUE_URL - String, included if report method successful, contains url to created issue

for Github


GithubService service = new GithubService("RepositoryOwner","RepositoryName");
service.create(null);//always null, url is defined internally

//first you must authenticate user using his Github username and password
Bundle b = service.auth("username", "password");
if(b.getBoolean(ReportingService.ACTION_SUCCESS)){}
  //authentication successful, you can send report
  b = service.report("report title", "report message");
  if(b.getBoolean(ReportingService.ACTION_SUCCESS)){
    //report sent successfully
    String url = b.getString(ReportingService.RESPONSE_ISSUE_URL);
  }
}

for Redmine


int projectId = 1; //your redmine project ID
RedmineService service = new RedmineService(projectId);
//specify your redmine installation URL
service.create("http://yourredmineurl/");

//first you must authenticate user using his Github username and password
Bundle b = service.auth("username", "password");
if(b.getBoolean(ReportingService.ACTION_SUCCESS)){}
  //authentication successful, you can send report
  b = service.report("report title", "report message");
  if(b.getBoolean(ReportingService.ACTION_SUCCESS)){
    //report sent successfully
    String url = b.getString(ReportingService.RESPONSE_ISSUE_URL);
  }
}
